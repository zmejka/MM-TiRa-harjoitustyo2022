### Asennus
----

1. Asennuskomento:

        poetry install

### Toiminnot
----

1. Käynnistys ja käyttö:

        poetry run invoke start


2. Testaus

        poetry run invoke test

3. Testikattavuus

        poetry run invoke coverage-report

4. Pylint testaus

        poetry run invoke lint
